Подробное описание

Класс, который используется на странице игры

Автор
Евгений
Версия
BETA 0.2.3
Дата
Ноябрь 2017 года

В этом классе содержатся все процедуры/функции, алгоритмы, которые необходимы для игры.

Предупреждения
Поскольку это BETA версия, то любая из процедур/функций/констант/переменных может в любой момент исчезнуть, если разработчик посчитает это нужным.
Необходимо сделать:
Отделение логики от дизайна. Дизайн может обращаться к логике, логика не может обращаться к дизайну.
Авторство
Все права на этот класс принадлежат разработчикам из AVB inc.
